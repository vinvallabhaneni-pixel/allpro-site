## GitHub Pages
To host on GitHub Pages:
1. Create a new repo (e.g. allpro-site)
2. Push all files to the main branch
3. Go to Settings → Pages → Source: main branch / root
4. Site will be live at https://yourusername.github.io/allpro-site/
